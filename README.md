# Installation for developments

```bash
$ yarn global add @nestjs/cli
```

```bash
$ yarn install --force
```

## Running the app
```bash
$ yarn build
```

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```
